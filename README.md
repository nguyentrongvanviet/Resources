## What’s inside
This folder groups links and references for study and reference. Each subfolder or file contains focused resources for a particular subject (notes, recommended readings, example code, datasets, and utilities).

## Topics covered
- Computer Science (algorithms, data structures, systems, competitive programming)
- Mathematics, Physics & Logic
- Machine Learning & AI
- Statistics & Data Analysis
- Web & App Development

## How to use
1. Browse the top-level folders to find a topic of interest.
2. Open `README.md` files inside subfolders for curated lists and quick-start notes.
3. Use the search feature in your editor to find specific topics or filenames.

## Contributing
Contributions are welcome! To add a resource:

1. Create a clear subfolder or update an existing one.
2. Add a short `README.md` explaining the resource with links and short annotations.
3. Open a pull request describing your addition.

Be concise and prefer canonical, well-maintained sources.
