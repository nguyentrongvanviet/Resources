## Books

### Algorithms and Data Structures
- **Introduction to Algorithms** by Thomas H. Cormen, Charles E. Leiserson, Ronald L. Rivest, and Clifford Stein
  - Comprehensive coverage of algorithms and data structures
  - Often referred to as "CLRS"
- **The Algorithm Design Manual** by Steven S. Skiena
  - Practical approach to algorithm design
  - Includes real-world examples and war stories
- **Data Structures and Algorithms in Python** by Michael T. Goodrich, Roberto Tamassia, and Michael H. Goldwasser
  - Python-based implementation of fundamental concepts

### Programming Languages
- **The C Programming Language** by Brian W. Kernighan and Dennis M. Ritchie
  - The definitive guide to C programming
  - Written by the creators of C
- **Effective Java** by Joshua Bloch
  - Best practices for Java programming
  - Essential for Java developers
- **Python Crash Course** by Eric Matthes
  - Beginner-friendly introduction to Python
  - Includes hands-on projects
- **JavaScript: The Good Parts** by Douglas Crockford
  - Focus on the elegant parts of JavaScript
  - Essential for web developers

### Software Engineering
- **Clean Code: A Handbook of Agile Software Craftsmanship** by Robert C. Martin
  - Best practices for writing maintainable code
  - Industry standard reference
- **Design Patterns: Elements of Reusable Object-Oriented Software** by Erich Gamma, Richard Helm, Ralph Johnson, and John Vlissides
  - The "Gang of Four" book on design patterns
  - Essential for object-oriented design
- **The Pragmatic Programmer** by Andrew Hunt and David Thomas
  - Timeless advice for software developers
  - Covers best practices and professional development

### Computer Systems
- **Computer Systems: A Programmer's Perspective** by Randal E. Bryant and David R. O'Hallaron
  - Understanding computer systems from a programmer's view
  - Covers hardware-software interface
- **Operating System Concepts** by Abraham Silberschatz, Peter B. Galvin, and Greg Gagne
  - Comprehensive guide to operating systems
  - Known as the "Dinosaur Book"
- **Computer Networking: A Top-Down Approach** by James F. Kurose and Keith W. Ross
  - Clear explanation of networking concepts
  - Top-down approach starting from applications

### Artificial Intelligence and Machine Learning
- **Artificial Intelligence: A Modern Approach** by Stuart Russell and Peter Norvig
  - Comprehensive AI textbook
  - Industry standard for AI courses
- **Pattern Recognition and Machine Learning** by Christopher M. Bishop
  - Mathematical foundations of machine learning
  - Covers Bayesian and frequentist approaches
- **Deep Learning** by Ian Goodfellow, Yoshua Bengio, and Aaron Courville
  - Comprehensive guide to deep learning
  - Available online for free
